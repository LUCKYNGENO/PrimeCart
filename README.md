# PrimeCart

PrimeCart is a Flask-based affiliate e-commerce platform.

## Features

- Admin dashboard
- Product management
- Search & categories
- Featured products
- Flash sales
- Wishlist
- Shopping cart
- Reviews & ratings
- Contact page
- About page

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- Jinja2

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```
